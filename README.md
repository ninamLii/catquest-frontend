# Cat Quest Readme

## setup
To store the secret api keys not in Code, I use dotenv to use a .env file. 
You need to add .env and add the apikeys. 
You can copy the .env.template and remove the .template

## Getting Started

1. Install dependencies with npm install
2. Create .env and add the env variables. An example is in env.template
3. Start your app with npm run dev
